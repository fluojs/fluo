Use `examples/minimal` when you want to see the smallest route and health-check shape before adding database, auth, or messaging packages.

## Shows

- Application bootstrap.
- Health/readiness routes.
- A small controller path.
- `@fluojs/testing` request helpers for app e2e tests.
