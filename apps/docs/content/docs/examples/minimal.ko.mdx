데이터베이스, 인증, 메시징 패키지를 추가하기 전에 가장 작은 route와 health-check shape를 보고 싶다면 `examples/minimal`을 사용하세요.

## 보여주는 것

- Application bootstrap.
- Health/readiness routes.
- 작은 controller path.
- App e2e test를 위한 `@fluojs/testing` request helper.
