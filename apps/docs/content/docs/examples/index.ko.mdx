예제는 가이드의 실행 가능한 동반 자료입니다. 단순한 happy path 스크린샷이 아니라, 문서화된 동작을 실제 app shape에서 가장 빠르게 증명하는 방법입니다.

## 예제 선택

| 예제 | 사용할 때 | 살펴볼 것 |
| --- | --- | --- |
| [Minimal](./minimal) | 가장 작은 runnable app이 필요할 때. | bootstrap, health/readiness routes, app e2e test |
| [RealWorld API](./realworld-api) | 실제 feature slice가 필요할 때. | modules, controllers, services, DTOs, validation, tests |
| [Auth JWT Passport](./auth-jwt-passport) | protected routes가 필요할 때. | token issuance, Passport strategy, profile route, auth tests |
| [Ops Metrics Terminus](./ops-metrics-terminus) | production 준비가 필요할 때. | health checks, readiness, metrics, operations routes |

## 예제 실행

저장소 root에서 실행합니다.

```bash
pnpm install
pnpm test -- --project examples
```

특정 예제만 확인하려면 해당 test file을 실행합니다. 예:

```bash
pnpm vitest run examples/minimal/src/app.test.ts
pnpm vitest run examples/auth-jwt-passport/src/app.test.ts
```

기대 결과: 선택한 test file이 통과하고 example page가 설명한 동작을 증명합니다.

## 예제 읽는 법

1. 먼저 example README를 읽습니다.
2. Module file을 열어 package composition을 봅니다.
3. Controller file을 열어 HTTP boundary를 봅니다.
4. Service/provider file을 열어 business logic을 봅니다.
5. Test file을 열어 예제가 약속하는 동작을 봅니다.
