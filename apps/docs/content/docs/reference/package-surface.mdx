The canonical package and runtime surface remains in the repository reference document: `docs/reference/package-surface.md`.

This docs-site page exists to make the official website navigation stable while keeping the source of truth in the governed repository docs.

## Families

- **Core**: decorators, dependency injection, configuration, and runtime orchestration.
- **HTTP**: request execution, validation, serialization, GraphQL, and OpenAPI.
- **Platform**: Fastify, Node.js, Express, Bun, Deno, and Cloudflare Workers adapters.
- **Persistence**: Prisma, Drizzle, Mongoose, Redis, and cache integrations.
- **Tooling**: CLI, testing, studio, and Vite integration.

Use [Package Selection](../guides/package-selection) when you need a task-oriented decision path.
