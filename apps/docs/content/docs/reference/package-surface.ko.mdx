정식 패키지 및 런타임 표면은 저장소의 레퍼런스 문서인 `docs/reference/package-surface.md`를 기준으로 유지됩니다.

이 문서 사이트 페이지는 공식 웹사이트 내비게이션을 안정적으로 제공하면서, source of truth는 거버넌스가 적용되는 저장소 문서에 남겨두기 위해 존재합니다.

## 계열

- **Core**: 데코레이터, 의존성 주입, 설정, 런타임 오케스트레이션.
- **HTTP**: 요청 실행, 검증, 직렬화, GraphQL, OpenAPI.
- **Platform**: Fastify, Node.js, Express, Bun, Deno, Cloudflare Workers 어댑터.
- **Persistence**: Prisma, Drizzle, Mongoose, Redis, cache 통합.
- **Tooling**: CLI, testing, studio, Vite 통합.

작업 중심 선택 경로가 필요하면 [패키지 선택](../guides/package-selection)을 사용하세요.
